# Travel App

This project is a travel application built with React and Vite. It includes features for searching flights, hotels, exploring destinations, and planning itineraries. The template has been customized beyond the default React+Vite boilerplate to incorporate application-specific APIs, components, and store logic.

## Project Structure

- `src/` - Main source code for the React application
- `src/api/` - Client-side API helpers
- `src/components/` - React components organized by feature
- `src/pages/` - Top-level page components
- `src/routes/` - Application routing setup
- `api/` - Serverless API routes for interacting with external services (flights, hotels, weather, etc.)
- `public/` - Static assets

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev` - Starts the development server
- `npm run build` - Builds the project for production
- `npm run preview` - Preview production build

## Technologies

- React
- Vite
